The technology behind the operator

Track 1

Live Agents

Hero Fitness uses the Gemini Live API for real-time voice dialogue. The Coach listens to you during the run and supports interruptions without breaking the GPS and Pace context, hosted for high performance on GCP Cloud Run.

Track 2

Creative Narrator

Mission Debriefs use Interleaved Multimodal data. Gemini narrates your run mixing epic narrative text with map snippets and stats in a single output stream.

Track 3

Interface Navigator

Through screenshots, the AI "sees" the Interactive Tactical Map UI. The Agent interprets red zones (enemies) and acts on the interface, alerting the runner about lost territories and reacting to visual HUD changes.
